background 속성은 태그의 배경을 지정하는 속성이며 앞으로 배우게 될 속성을 가집니다. 세부적인 속성들을 한번에 쓸 수 있습니다.

  1. background-color : 배경색
  2. background-image : 배경 이미지
  3. background-repeat : 배경 이미지 반복 여부
  4. background-position : 배경 이미지 위치 속성

CSS 배경색 관련 속성

red(#ff0000)
orange(#ffa500)
yellow(#ffff00)
green(#008000)
blue(#0000ff)
#000080
purple(#800080)
